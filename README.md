Proyek ini merupakan bagian dari penelitian skripsi berjudul:
"IDENTIFIKASI WAJAH BELL’S PALSY DAN STROKE MENGGUNAKAN SVM DENGAN OPTIMASI GRID SEARCH"
oleh Chelsea Effendi

Aplikasi ini menggunakan Computer Vision untuk mengekstraksi fitur numerik dari wajah, kemudian melakukan klasifikasi apakah wajah tersebut menunjukkan gejala Stroke atau Bell’s Palsy, menggunakan algoritma Support Vector Machine (SVM) kernel RBF dengan optimasi Grid Search.

Data splitting menggunakan 3 rasio, dimana rasio paling optimal adalah 90:10, sehingga model yang diimplementasikan dalam situs web adalah 'SVM_GD_91_RBF_best_model'.
